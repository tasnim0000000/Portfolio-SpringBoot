# Where to put project screenshots

Put a screenshot for each project in this folder, named exactly:

| File name              | Project      | What to screenshot                              |
|-------------------------|--------------|--------------------------------------------------|
| `pawpal.jpg`            | PawPal       | The JavaFX app UI — e.g. the login screen or one of the dashboards (Buyer/Seller/Admin) |
| `studyflow.jpg`         | StudyFlow    | The dashboard or Kanban board in the browser      |
| `coxs-canvas.jpg`       | Cox's Canvas | A rendered frame of the OpenGL beach scene (day or sunset view looks best) |

- Format: `.jpg` (rename + update the matching `<img src="...">` in `index.html`
  under the "PROJECTS" section if you use `.png`/`.webp` instead)
- Recommended size: roughly **800×450px** (16:9), landscape orientation — this
  matches the card's cover area
- If a file is missing, that card just falls back to showing the project name
  as text on a soft gradient background — nothing breaks.
