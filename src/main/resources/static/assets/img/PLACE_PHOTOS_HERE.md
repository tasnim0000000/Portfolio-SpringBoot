# Where to put the profile photo

Put **one photo of Tasnim** in this exact folder, named exactly:

```
assets/img/profile.jpg
```

- Format: `.jpg` (or rename it and update the `src` in `index.html` around the
  "about-portrait-wrap" section if you'd rather use `.png`/`.webp`)
- Recommended size: at least **800×1000px**, portrait orientation (taller than wide)
- A simple headshot or half-body photo with good lighting works best — the frame
  crops to fill a tall rectangle, so keep her face/upper body centered.
- If this file is missing, the site does NOT break — it just falls back to the
  "TA" initials placeholder that's already on the live site.

Project screenshots go in `assets/img/projects/` — see the README there.
